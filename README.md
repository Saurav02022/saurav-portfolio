# Saurav Kumar — Portfolio

Personal portfolio at [saurav02022-portfolio.vercel.app](https://saurav02022-portfolio.vercel.app). Single-page, dark, fully static — Next.js (App Router) + TypeScript + Tailwind CSS v4, deployed on Vercel.

## Stack

- **Next.js 16** (App Router, static output) with `next/font` (Hanken Grotesk, Newsreader, JetBrains Mono)
- **Tailwind CSS v4** — design tokens as CSS variables in [app/globals.css](app/globals.css)
- **TypeScript** throughout

## Editing content

All copy lives in [lib/portfolio-data.tsx](lib/portfolio-data.tsx) (typed by [lib/types.ts](lib/types.ts)) — hero, experience, case studies, about, contact, socials, and the résumé URL. Components under `components/` only render that data; content changes never require component edits.

To serve the résumé locally, drop `Saurav-Kumar-Resume.pdf` into `public/` and point `RESUME_URL` at `/Saurav-Kumar-Resume.pdf`.

## Development

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build (fully static)
npm run lint
```

## SEO

- Metadata via the Next.js Metadata API ([app/layout.tsx](app/layout.tsx)); canonical URL from `NEXT_PUBLIC_SITE_URL` (defaults in [lib/site-config.ts](lib/site-config.ts))
- `robots.txt` and `sitemap.xml` generated by [app/robots.ts](app/robots.ts) and [app/sitemap.ts](app/sitemap.ts)
- JSON-LD (Person, WebSite, SoftwareSourceCode, BreadcrumbList) in [components/structured-data.tsx](components/structured-data.tsx)
- OG share card and favicons generated at build from design tokens ([app/opengraph-image.tsx](app/opengraph-image.tsx), [app/icon.tsx](app/icon.tsx)) — fonts committed in `assets/fonts/`
- Google verification via `NEXT_PUBLIC_GOOGLE_SITE_VERIFICATION`; visitor stats via Vercel Analytics

Brand voice and cross-surface copy (LinkedIn, GitHub bio, hero line) live in [BRAND.md](BRAND.md).
