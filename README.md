## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/Saurav02022/saurav-portfolio.git
cd saurav-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

**Environment Setup:**

Create `.env.local`:
```env
VITE_GITHUB_USERNAME=Saurav02022
```
